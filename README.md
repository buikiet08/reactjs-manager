# Project setup

Bước 1: Copy .env.example -> .env và cung cấp các env cần thiết

Bước 2: Cài đặt các package

> yarn || npm i

Bước 3: Run dự án

> yarn dev || npm run dev

Bước 4. Truy cập 

> http://localhost:3000
